- Cài đặt cấu trúc dữ liệu QUEUE sử dụng mảng, liên kết đơn.

- Project Contactlist With Menu: Quản lý danh sách danh bạ sử dụng liên kết đơn. Cung cấp các chức năng
  + Nạp dữ liệu từ file.
